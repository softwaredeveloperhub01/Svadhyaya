# Svādhyāya (स्वाध्याय) — Self-Study Mirror

> "Astrology is not a set of predictions; it is a mirror for self-observation."

Svādhyāya is an open-source, local-first Vedic application designed for the **Modern Skeptical Seeker**. In a market filled with data-hungry and fear-driven astrology apps, Svādhyāya stands as a "Crystal Glass" of trust.

### 🎯 The Mission
To empower individuals with 100% accurate astronomical data (Drik Ganita) to help them understand their internal "emotional weather" and cultivate self-awareness.

### 🛡️ The Proper & Mature Manifesto
1. **Zero Data Collection:** Your birth details never leave your device. We use no servers.
2. **No Fear-Mongering:** We translate planetary transits into growth strategies, not scary "fortunes."
3. **Scientific Precision:** Powered by the Swiss Ephemeris (NASA-level math).
4. **$0 Cost Forever:** Built by the community, for the community.

### 🛠️ Technology Stack
* **Language:** Pure JavaScript (Vanilla JS)
* **Engine:** Swiss Ephemeris (WASM) via `@swisseph/browser`
* **Vedic Logic:** `panchangam-js`
* **UI:** Clean HTML5/CSS3 (Zen Aesthetic)
